# Learning Data, One Step at a Time - Jurnal Belajar Pribadi

Website ini adalah proyek portofolio dan jurnal digital personal yang mendokumentasikan perjalanan belajar saya sebagai mahasiswi Sistem Informasi (FILKOM UB) menuju impian menjadi seorang Data Analyst. Proyek ini dibangun untuk memenuhi Tugas Individu mata kuliah Desain Antarmuka Pengguna.

## 🎨 Penjelasan Desain UI/UX (Prinsip Dasar)

Website ini dirancang secara matang dengan menerapkan prinsip-prinsip utama antarmuka pengguna:

### 1. Target Pengguna (Target Audience)
- Dosen dan Asisten Praktikum mata kuliah Desain Antarmuka Pengguna.
- Rekruter/Praktisi industri di bidang Data Science/Analytics.
- Rekan sesama mahasiswa yang mencari referensi jurnal belajar.

### 2. Konsistensi Desain (Consistency)
- **Warna:** Menggunakan palet warna bertema alam/sage yang menenangkan (Sage Green `#A7B194`, Off-White `#FBFBF9`, dan Dark Slate `#2D3129`) untuk menciptakan kesan jurnal/scrapbook digital yang estetik dan profesional.
- **Tipografi:** Menggunakan kombinasi font *Playfair Display* (Serif) untuk memberikan hierarki visual yang kuat pada judul/heading, serta *DM Sans* (Sans-Serif) untuk kejelasan keterbacaan pada teks paragraf body.

### 3. Tata Letak Rapi & Responsif (Layout & Responsiveness)
- Layout diatur menggunakan CSS Grid dan Flexbox untuk memastikan kerapian susunan komponen.
- Dilengkapi dengan optimasi `@media screen` sehingga tampilan website otomatis menyesuaikan (responsif) saat diakses melalui perangkat desktop, tablet, maupun layar *handphone* (mobile) tanpa ada teks yang terpotong.

### 4. Navigasi yang Jelas (Navigation)
- Memiliki komponen *Fixed Navbar* di bagian atas yang selalu menetap saat di-scroll, memudahkan pengguna berpindah antar-bagian halaman (Beranda, Tentang, Timeline, Projek, Skill) secara instan.

## 🛠️ Teknologi yang Digunakan
- **HTML5:** Struktur semantik dokumen website.
- **CSS3:** Kustomisasi layout, pewarnaan, scrapbook styling, dan animasi melayang (*floating animation*).
- **FontAwesome:** Library ikon visual pada timeline dan footer.
- **GitHub Pages:** Layanan hosting gratis untuk meluncurkan website ke publik.
